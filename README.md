# Daily Builds

A 30-minute-a-day practice in shipping. One small thing every day, varied across languages and domains. The point is the habit; the side effect is a record of how I think and what I'm curious about.

## Rotation

- **Mon** — Python utility
- **Tue** — Frontend mini-app
- **Wed** — CLI tool
- **Thu** — Data viz / notebook
- **Fri** — API / scraper
- **Sat** — Refactor or extend
- **Sun** — Learn one thing
